# Contributing to SpendMate

We welcome contributions! Here's how you can help:

## Quick Start

1. Fork the repository
2. Clone it: `git clone https://github.com/farhan10ansari/SpendMate.git`
3. Create a branch: `git checkout -b feature/your-feature`
4. Make changes and test them
5. Commit: `git commit -m 'Add feature'`
6. Push and open a Pull Request

## Guidelines

- Run `yarn lint` before submitting
- Keep PRs focused and reasonably sized
- Reference related issues in your PR
- We review PRs within 24-48 hours
- Follow the Code of Conduct below

## Have Questions?

- **Email:** spendmate.assist@gmail.com
- **GitHub Issues:** [Report bugs](https://github.com/farhan10ansari/spendmate/issues)
- **Telegram:** [Join community](https://t.me/farhan10ansari_spend_mate_disc)

# Code of Conduct

## Our Pledge

We are committed to providing a welcoming and inclusive environment for all contributors and community members, regardless of age, body size, disability, ethnicity, gender identity, level of experience, education, socioeconomic status, nationality, personal appearance, race, religion, or sexual identity and orientation.

## Our Standards

**Positive behavior includes:**

- Being respectful and kind to others
- Welcoming feedback and diverse viewpoints
- Accepting responsibility and apologizing when needed
- Focusing on what's best for the community

**Unacceptable behavior includes:**

- Harassment, discrimination, or hateful speech
- Trolling, insults, or derogatory comments
- Personal attacks or intimidation
- Publishing others' private information without consent
- Any other inappropriate professional conduct

## Enforcement

Violations of this code of conduct may result in:

- Warning or temporary suspension
- Permanent removal from the community

## Reporting

If you witness or experience a violation, report it to: **spendmate.assist@gmail.com**

Include details about the incident and any relevant context. All reports are handled confidentially.
