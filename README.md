## ğŸ“¥ Installation

### Prerequisites
- Node.js v14+ or Python 3.8+
- npm or yarn package manager
- Git

### Clone Repository
```

git clone https://github.com/your-username/spendmate.git
cd spendmate

```

### Install Dependencies
```

npm install

# or

yarn install

```

### Environment Setup
```

cp .env.example .env

# Update .env with your configuration
```
